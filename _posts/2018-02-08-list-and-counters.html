Объявляем элемент списка

Элемент списка - любой элемент с display: list-item; или display: inline-list-item;.

Элемент списка генерирует псевдо-элемент ::marker.

Дополнительно, элементы списка автоматически инкрементируют специальный счетчик элементов списка.

Значение inline-list-item приводит к тому, что вычисленное значение list-style-position равно inside.

Свойства list-style-image и list-style-type

Содержимое псевдо-элемента ::marker может генерироваться автоматически. Свойства list-style-image и list-style-type управляют тем, как это содержимое генерируется.

Свойство list-style-image определяет изображение, которое будет использоваться в качестве содержимого маркера по умолчанию.

Свойство list-style-image
Значения image | none
По умолчанию none
Применяется к элементы списка
Наследуется да

image - если изображение валидно, оно становится содержимым маркера элемента-списка. Иначе содержимое маркера определяется list-style-type.

none - содержимое маркера задается свойством list-style-type.

Свойство list-style-type
Значения counter-style |string | none
По умолчанию none
Применяется к элементы списка
Наследуется да

counter-style - содержимое маркера - стиль счетчика. CSS Counter Styles Level 3

string - в качестве содержимого маркера используется строка string.

none - у маркера нет содержимого.

list-style-position

Свойство определяет позицию маркера.

Свойство list-style-position
Значения inside | outside
По умолчанию none
Применяется к элементы списка
Наследуется да

list-style

Свойство list-style
Значения list-style-type || list-style-position || list-style-image

Псевдо-элемент ::marker

Псевдо-элемент ::marker можно стилизовать. Создается он только для элементов списка.
